.iface_api.init

触发时机：系统启动阶段（内核初始化完成，协议栈启动时）。

调用者：net_if.c 中的接口初始化代码。

作用：将驱动实例与网络接口（struct net_if）绑定。

你的任务：在这个函数里设置 MAC 地址、初始化内部信号量（Semaphores），并告知系统该接口的最大传输单元（MTU）。

```
net_if_set_link_addr(iface, ctx->mac_addr,sizeof(ctx->mac_addr),NET_LINK_ETHERNET);

ethernet_init(iface);

net_if_carrier_off(iface);
```



调用位置:
`net_if_init`
中的`init_iface`
```
STRUCT_SECTION_FOREACH(net_if, iface) {
	...
	init_iface(iface);
	...
	net_stats_prometheus_init(iface);
	if_count++;
}
```
