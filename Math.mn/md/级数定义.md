在初等数学中，我们习惯于通过直角三角形或单位圆来定义 $\sin x$。但在高等数学（分析学）中，这种定义存在**逻辑循环**和**局限性**。

以下是为什么要从“几何定义”转向“级数定义”的深层逻辑，以及这种定义带来的数学威力。

---

### 1. 为什么要抛弃几何定义？（严谨性的必要）

初等数学的定义依赖于“长度”和“角度”的概念。但在建立严格的微积分体系时，我们会遇到以下问题：

- **逻辑循环：** 几何定义通常依赖于“圆周长”或“弧长”。而弧长的计算本身需要积分，积分又需要导数，$\sin x$ 的导数证明在几何框架下往往需要用到 $\lim_{x \to 0} \frac{\sin x}{x} = 1$ 这个极限，而这个极限的几何证明通常预设了面积或弧长的结论。这在逻辑上是不够闭环的。
    
- **无法推广：** 几何定义只能处理**实数**（角度）。你无法画出一个角度为 $i$（虚数单位）的三角形。
    

### 2. 级数定义：纯粹的解析定义

高等数学通过幂级数直接定义函数：

$$\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \dots + \frac{(-1)^n x^{2n+1}}{(2n+1)!} + \dots$$

$$\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \dots + \frac{(-1)^n x^{2n}}{(2n)!} + \dots$$

**这种定义的优势：**

- **收敛性：** 利用比值判别法（Ratio Test）可以证明，这两个级数对**任何复数 $x$** 都是绝对收敛的。这意味着三角函数从此有了坚实的算术基础。
    
- **导数直接导出：** 无需几何直观，直接对级数进行逐项求导，立刻就能得到 $(\sin x)' = \cos x$ 和 $(\cos x)' = -\sin x$。
    

### 3. 从实数到复数：欧拉公式的诞生

这是级数定义最伟大的贡献。如果我们把虚数 $iz$ 代入指数函数 $e^z$ 的泰勒展开式：

$$e^{iz} = 1 + (iz) + \frac{(iz)^2}{2!} + \frac{(iz)^3}{3!} + \frac{(iz)^4}{4!} + \dots$$

由于 $i^2 = -1, i^3 = -i, i^4 = 1$，我们将项重新组合：

$$e^{iz} = \left( 1 - \frac{z^2}{2!} + \frac{z^4}{4!} - \dots \right) + i \left( z - \frac{z^3}{3!} + \frac{z^5}{5!} - \dots \right)$$

这就是：

$$e^{iz} = \cos z + i \sin z$$

专家视角的洞察：

通过级数定义，我们发现三角函数其实是指数函数的另一种表现形式。它们本质上都是一种幂级数，只是系数选取不同。

### 4. 扩展应用：复数域下的三角函数

有了级数定义，我们可以计算一些“反直觉”的东西：

- **虚数角度：** 我们可以计算 $\sin(i)$。通过级数或欧拉公式可知，$\sin(iz) = i \sinh(z)$（双曲正弦）。这在波动方程和量子力学中处理“衰减波”时至关重要。
    
- **无界性：** 在实数域，$|\sin x| \le 1$；但在复数域，三角函数是**无界**的。
    

### 5. 重新找回几何：为什么它还是圆？

你可能会问：既然放弃了三角形，那它和圆还有关系吗？

有。当我们定义了 $\sin x$ 和 $\cos x$ 之后，我们可以通过级数乘法严格证明：

$$\sin^2 x + \cos^2 x = 1$$

这个代数恒等式说明，点 $(\cos x, \sin x)$ 永远落在平面直角坐标系的单位圆上。

总结：

高等数学通过级数定义，将三角函数从“测量工具”升级为了“解析算子”。它不再依赖于你手里的量角器，而是成为了一个可以通过无限次加减乘除精确定义的数学对象。这使得三角函数能完美地融入复分析、微分方程等更高端的领域。
